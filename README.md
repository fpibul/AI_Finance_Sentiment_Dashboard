AI Financial News Sentiment Analyzer ---------

A real-time AI dashboard that pulls the latest financial news for any stock or keyword and analyzes sentiment using transformer-based NLP models. Built with Python, HuggingFace Transformers, Streamlit, and NewsAPI.

THIS APP:

- Pulls real-time financial news using NewsAPI
- Extracts the top 5 headlines for any stock/company (e.g., Tesla, Apple, Federal Reserve)
- Uses a transformer model to classify each headline as Positive or Negative

and CALCULATES:

- Number of positive vs negative articles
- Market sentiment balance
- Automatically generates a sentiment pie chart
- Displays all results in a clean, interactive Streamlit dashboard
